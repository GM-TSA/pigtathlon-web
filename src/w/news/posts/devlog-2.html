New Run & Gun Boss

We created a final boss for our run & gun minigame. The boss is a tiger king which must be killed to complete the level.

New Run & Gun Assets

We added some new assets to the run & gun minigame like a hut and a river bridge (with some pretty cool water effects).



Local Multiplayer

We added back local multiplayer to the run & gun minigame since that is one of our main goals in creating this game. It does get a bit laggy but we have plans on making that run better

Run & Gun Tutorial

We added a quick tutorial to the beginning of the Run & Gun Minigame for new players. This tutorial may be streamlined in the future.
